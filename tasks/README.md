# Tasks

Task folders store durable execution context that is more specific than the
repo-wide docs in `docs/`.

Each task folder should contain:

- `discovery-report.md`
- `needs-manifest.md`
- `contract.md`
- `plan.md`
- `implement.md`
- `documentation.md`

Optional files:

- `requirements-traceability.md`
- `business-process-map.md`
- `uat-matrix.md`
